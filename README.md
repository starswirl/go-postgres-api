# go-postgres-api

go + gin + postgresのDocker環境

## 立ち上げ方

1. [Go環境構築](doc/Go環境構築.md)を読む
2. .env.sampleを参考に.env作る

3. 下記コマンド

```bash
cd server
go get
cd ..
docker-compose up
```
